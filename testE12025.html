4. La examenul pentru șoferi auto s-au înregistrat N persoane. Se știe că punctajul minim care trebuie acumulat de o persoană ca să susțină examenul este egal cu P.

Sarcina: Scrieți un program care determină numărul de persoane care au susținut examenul.

Intrare. De la tastatură se citesc două numere întregi: N – numărul de persoane înregistrate la examen și P – punctajul minim necesar pentru a susține examenul. Apoi se citesc N numere întregi – punctajul acumulat de fiecare din persoanele înregistrate la examen.

Ieșire: . La ecran se afișează un singur număr întreg – numărul de persoane care au susținut examenul.

Raspuns corect:

#include <iostream>
using namespace std;
int main() {
int N, P, count = 0;
cin >> N >> P;
for (int i = 0; i < N; i++) {
int score;
cin >> score;
if (score >= P) {
count++;
}
}
cout << count << endl;
return 0;
}

Notă. Se admit oricare alte variante corecte.
