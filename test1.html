4. De la tastatură se citesc 20 de coduri ale tabelului ASCII extins - numere întregi între 0 și 255.

Sarcina: Scrieți un program care determină câte dintre codurile citite corespund codului caracterului ’+’.

Ieșire: La ecran se afișează un singur număr întreg – numărul de coduri ale tabelului ASCII extins, citite de la tastatură, care corespund codului caracterului ‘+’.

Raspunsul corect:

#include <iostream>
int main() {
const int n = 20;
int count = 0;

std::cout << "Introduceți 20 de coduri ASCII (între 0 și 255):\n";
for (int i = 0; i < n; ++i) {
int code;
std::cin >> code;
if (code == 43) {
++count;
}
}
std::cout << count << std::endl;
return 0;
}
