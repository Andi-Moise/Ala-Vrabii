Exerciții tip 4
4. De la tastatură se citesc 20 de coduri ale tabelului ASCII extins - numere întregi între 0 și 255.

Sarcina: Scrieți un program care determină câte dintre codurile citite corespund codului caracterului ’+’.

Ieșire: La ecran se afișează un singur număr întreg – numărul de coduri ale tabelului ASCII extins, citite de la tastatură, care corespund codului caracterului ‘+’.

Raspunsul corect:

#include <iostream>
int main() {
const int n = 20;
int count = 0;

std::cout << "Introduceți 20 de coduri ASCII (între 0 și 255):\n";
for (int i = 0; i < n; ++i) {
int code;
std::cin >> code;
if (code == 43) {
++count;
}
}
std::cout << count << std::endl;
return 0;
}

Punctaj: 8

4. La un concurs de erudiție doi elevi au acumulat punctaje care reprezintă numere de trei cifre. Elevii au observat că punctajul acumulat de fiecare dintre ei este un număr egal cu răsturnatul celuilalt.

Sarcina: Scrieți un program care cunoscând punctajul N acumulat de primul elev determină punctajul acumulat de al doilea elev.

Intrare. e. De la tastatură se citește un număr întreg N - punctajul acumulat de primul elev

Ieșire: La ecran se afișează un singur număr întreg – răsturnatul numărului N.

Exemplu: Numărul 529 este răsturnatul numărului 925

Raspuns corect:

#include <iostream>
using namespace std;
int main() {
int N, P, count = 0;
cin >> N >> P;
for (int i = 0; i < N; i++) {
int score;
cin >> score;
if (score >= P) {
count++;
}
}
cout << count << endl;
return 0;
}

Notă. Se admit oricare alte variante corecte.

Punctaj: 8

4. Tabelarea unei funcții de o variabilă reprezintă generarea valorilor funcției f(x) pentru toate valorile argumentelor care parcurg un domeniu de valori de la x1 până la x2 cu pasul ∆x, adică x1, x1+∆x, x1+2*∆x, x1+3*∆x, ... , x2.

Sarcina: Scrieți un program care tabelează funcția f(x)=3*sin(x2+5)pe segmentul [x1;x2] cu pasul ∆x.

Intrare: . De la tastatură se citesc trei numere reale x1, x2 și ∆x .

Ieșire: La ecran se afișează pe rânduri separate câte două numere reale separate prin spațiu: valoarea argumentului x și valoarea funcției f(x) generate de tabelarea acesteia.

Notă: Se admite, că x1 < x2.

Raspunsul corect:

#include <iostream>
#include <cmath>
#include <iomanip>
using namespace std;
int main() {
double x1, x2, delta_x;
cout << "Introduceți x1: ";
cin >> x1;
cout << "Introduceți x2: ";
cin >> x2;
cout << "Introduceți ∆x: ";
cin >> delta_x;
if (x1 >= x2) {
cout << "x1 trebuie să fie mai mic decât x2." << endl;
return 1;
}
cout << fixed << setprecision(6);
for (double x = x1; x <= x2 + 1e-9; x += delta_x) {
double fx = 3 * sin(x * x + 5);
cout << x << " " << fx << endl;
}
return 0;
}

Punctaj: 8

4. La examenul pentru șoferi auto s-au înregistrat N persoane. Se știe că punctajul minim care trebuie acumulat de o persoană ca să susțină examenul este egal cu P.

Sarcina: Scrieți un program care determină numărul de persoane care au susținut examenul.

Intrare. De la tastatură se citesc două numere întregi: N – numărul de persoane înregistrate la examen și P – punctajul minim necesar pentru a susține examenul. Apoi se citesc N numere întregi – punctajul acumulat de fiecare din persoanele înregistrate la examen.

Ieșire: . La ecran se afișează un singur număr întreg – numărul de persoane care au susținut examenul.

Raspuns corect:

#include <iostream>
using namespace std;
int main() {
int N, P, count = 0;
cin >> N >> P;
for (int i = 0; i < N; i++) {
int score;
cin >> score;
if (score >= P) {
count++;
}
}
cout << count << endl;
return 0;
}

Notă. Se admit oricare alte variante corecte.
