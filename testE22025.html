4. La un concurs de erudiție doi elevi au acumulat punctaje care reprezintă numere de trei cifre. Elevii au observat că punctajul acumulat de fiecare dintre ei este un număr egal cu răsturnatul celuilalt.

Sarcina: Scrieți un program care cunoscând punctajul N acumulat de primul elev determină punctajul acumulat de al doilea elev.

Intrare. e. De la tastatură se citește un număr întreg N - punctajul acumulat de primul elev

Ieșire: La ecran se afișează un singur număr întreg – răsturnatul numărului N.

Exemplu: Numărul 529 este răsturnatul numărului 925

Raspuns corect:

#include <iostream>
using namespace std;
int main() {
int N, P, count = 0;
cin >> N >> P;
for (int i = 0; i < N; i++) {
int score;
cin >> score;
if (score >= P) {
count++;
}
}
cout << count << endl;
return 0;
}

Notă. Se admit oricare alte variante corecte.
